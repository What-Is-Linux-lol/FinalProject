How I Created This Website

This website was built as a personal project to showcase my background and interests. I wanted it to be easy to navigate and a good foundation for learning web development.

Steps I Took

  • Started with a basic index.html file to serve as the homepage.
  • Created additional pages: aboutme.html, contact.html, and thiswebsite.html.
  • Added a shared style.css file to keep the design consistent across all pages.
  • Built a simple navigation bar with tabs so visitors can move between pages easily.
  • Uploaded the files to my GitHub repository and enabled GitHub Pages to host the site.
  • Organized assets like images into a dedicated images/ folder.
